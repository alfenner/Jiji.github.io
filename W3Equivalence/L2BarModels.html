Bar Model Equivalence

Lesson Overview

Warm Up: Bar Model Relationships
  • play_circle_filledPart 1: Students use the bar model tool to find relationships and patterns (4 min).
  • groupPart 2: Project bar model prompt while students record and discuss relationships and patterns (4min).record_voice_over
  • groupPart 3: Discuss a couple of relationships students found as a whole class. (4 min).record_voice_over
Activity One: Model Equivalent Fractions
  • play_circle_filled
    Part 1: Project model with equivalent fractions task while students solve independently(4 min ).
  • groupPart 2: Discuss equivalent bar models for problems (4 min).record_voice_over
Activity Two: Equivalent Fraction Equations
  • assignmentPart 1: Project write equivalent fractions task while students solve independently (4 min ). in pairs (7 min)
  • groupPart 2: Discuss connections between bar model and equation representations (4 min).airplay
Activity Three: Missing Fraction Equations
  • assignmentPart 1: Project missing fraction equations task while students solve missing fraction equations (4 min ).
  • groupPart 2: Discuss strategies for finding missing fractions (4 min).airplay
Conclusion: Bar Model Problem Solving
  • facePart 1: Project conclusion slide while students solve independently (4 min).
  • facePart 2: Discuss strategies for finding the value of bars not labeled (4 min).
