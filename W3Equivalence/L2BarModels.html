Bar Model Equivalence

Lesson Overview

Warm Up: Bar Model Relationships
  • personPart 1a: Students use the bar model tool to find relationships and patterns (5 min) face
  • groupPart 1b: Project exploration prompt for students to find relationships and patterns (4min) airplay
  • assignmentPart 2: Students use Explore Relationships and Patterns problem sheet to record their observations (5min)
  • groupPart 3: Discuss a couple of relationships students found as a whole class (5 min)
Activity One: Model Equivalent Fractions
  • group
    Part 1a: Project prompt for students to write equations for equivalent fractions.airplay
  • assignmentPart 1b: Students write equations to represent equivalent fractions build in activity one. Model Equivalent Fractions pt 1 problem sheet (5 min)
  • groupPart 2: Discuss equivalent bar models for problems (4 min)
Activity Two: Equivalent Fraction Equations
  • groupPart 1a: Project prompt for students to write equations for equivalent fractions airplay
  • assignmentPart 1b: Students write equations to represent equivalent fractions build in activity one. Model Equivalent Fractions pt 2 problem sheet (5 min)
Activity Three: Missing Fraction Equations
  • groupPart 1a: Project prompt for students to find the missing fraction in an equation airplay
  • assignmentPart 1b: Students solve missing fraction equations in Model Equivalent Fractions pt 3 problem sheet (5 min)
Conclusion: Bar Model Problem Solving
  • personPart 1a: Project conclusion slide while students solve independently (4 min)
  • assignmentPart 1b: Sudents use the bar model tool to solve Model Equivalent Fractions Conclusion problem sheet (5min)
  • groupPart 2: Discuss strategies for finding the value of bars not labeled (4 min)
